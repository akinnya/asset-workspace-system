# 素材与外部资源说明

本文件用于说明仓库中的视觉素材、截图与外部资源边界，帮助维护者在公开发布前完成可再分发核对。

## 仓库内已跟踪的视觉资源

当前仓库包含以下几类视觉文件：

- `docs/readme/` 下的 README 截图与架构图
- `frontend/src/assets/` 下的 logo、头像、登录页背景与吉祥物素材
- `frontend/public/` 下的默认头像等前端静态资源

## 发布前必须确认的事项

在把仓库设置为公开之前，请逐项确认：

- 上述素材是否为项目维护者原创，或已获得明确的再分发授权
- 截图中是否包含个人信息、真实账号、内部数据或不可公开界面
- 图标、字体、背景纹理、插画与品牌标识是否具备与仓库发布方式兼容的授权条件
- 若某个素材只允许展示、不允许再分发，应在公开前替换或移除

## 外部运行时资源

仓库运行时还会访问以下外部资源或第三方服务：

- Google Fonts：用于前端字体加载
- Transparent Textures：用于工作区页面背景纹理
- Bing 图片接口：用于批量抓图预览链路
- 阿里云 OSS / 邮件服务：属于可选集成能力，使用时需自行提供凭据并遵守对应服务条款

这些资源并不等同于“随仓库一起授权分发的文件”。公开仓库前，应明确区分：

- 哪些文件已随仓库分发
- 哪些能力只是运行时访问外部服务
- 哪些第三方资源需要单独列出来源与许可

## 建议的维护方式

- 若素材全部为原创，可在本文件补充“原创确认”记录。
- 若存在第三方素材，请在本文件追加来源、许可类型、获取时间和使用范围。
- 若授权状态无法确认，建议在公开前先替换为可确认授权的资源。
